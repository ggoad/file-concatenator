This is a file concatenator, suitable for combining multiple files into one. 

It is suitable for remote files, using the http and https protocols. 

Specify an output file name, or just copy the results upon submission.

Omit the protocal from a file-list member to find a local file.

This file requires a PHP environment to run. 



DO NOT USE THIS ON A LIVE SERVER

It is vulnerable to file-traversal attacks, and likely script-injection attacks. It is completely possible for this script to overwrite critical files.



DO USE THIS

To pre-calculate a concatonated file to reduce your server requests.


INSTALATION

Copy to your php server, visit the page, fill out the form, and submit. That's it!
